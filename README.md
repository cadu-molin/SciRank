"# SciRank" 

Projeto feito em JavaScript, usando as seguintes ferramentas:
- Nodemon "Usado para hotreload"
- Migrations(Sequalize)
- Postgres
- Express
- Handlebars

- .env:
    POSTGRES_HOST=127.0.0.1
    POSTGRES_PORT=5432
    POSTGRES_USER=postgres
    POSTGRES_PW=postgres
    POSTGRES_DB=SciRank
    API_URL=127.0.0.1:8081
